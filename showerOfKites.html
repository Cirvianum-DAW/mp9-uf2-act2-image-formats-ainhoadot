Question 1: Which format is the most optimal for this type of image and why?

Answer: The JPEG (High) format is the most optimal for this type of image, as it has a relatively low file size and maintains adequate image quality.

Question 2: With what compression quality does the image not lose quality?

Answer: With the compression quality "High" the image does not lose quality noticeably.

Question 3: Can we further reduce its size without losing quality?

Answer: We can further reduce its size without losing quality, but this may involve a loss of detail and definition as we increase the compression.

Question 4: What would happen if we save the image several times in JPEG format?

Answer: If we save the image multiple times in JPEG format, each time you save it, a new lossy compression occurs, which can lead to a progressive degradation of the image quality with each compression. This can cause visual artifacts and a general decrease in image quality.
